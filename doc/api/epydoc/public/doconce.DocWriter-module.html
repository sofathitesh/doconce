Module doconce.DocWriter

DocWriter is a tool for writing documents in ASCII, HTML, LaTeX, Doconce, and other formats based on input from Python datastructures.

The base class _BaseWriter defines common functions and data structures, while subclasses HTML, Doconce, etc. implement (i.e., write to) various formats.

This module works, but is unifinished and needs documentation!
